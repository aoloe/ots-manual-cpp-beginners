# Notes

## Install

### Linux

You need a compiler
- `g++` or
- `clang`

A text editor:
- `GEdit` on Gnome / `Kate` on KDE
- `leafpad` (GTK, very basic)
- `sublime text`
- `atom` (?)
- `vim` / `emacs`
- `jedit` (multiplatform)

#### Debian / Ubuntu

~~~
sudo apt-get install g++
~~~

### Windows

- Visual Studio
- MinGW GCC
- (Eclipse ?)
- cygwin
- virtualbox with a linux distributino
